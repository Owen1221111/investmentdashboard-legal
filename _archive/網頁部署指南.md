# 隱私政策和使用條款網頁部署指南

本指南將說明如何使用 GitHub Pages 免費託管您的隱私政策和使用條款。

## 方法一：使用 GitHub Pages（推薦）

### 步驟 1：創建 GitHub 帳號
1. 前往 https://github.com
2. 點擊 "Sign up" 註冊帳號
3. 免費帳號即可使用 GitHub Pages

### 步驟 2：創建新的 Repository
1. 登入 GitHub 後，點擊右上角的 "+" 按鈕
2. 選擇 "New repository"
3. Repository 名稱：`投資看板-隱私政策`（或任何您喜歡的名稱）
4. 設定為 "Public"（公開）
5. 勾選 "Add a README file"
6. 點擊 "Create repository"

### 步驟 3：上傳檔案
1. 在 repository 頁面，點擊 "Add file" > "Upload files"
2. 上傳以下檔案：
   - `privacy-zh.html`（隱私政策中文版）
   - `privacy-en.html`（隱私政策英文版）
   - `terms-zh.html`（使用條款中文版）
   - `terms-en.html`（使用條款英文版）
   - `index.html`（首頁）
3. 點擊 "Commit changes"

### 步驟 4：啟用 GitHub Pages
1. 在 repository 頁面，點擊 "Settings"
2. 在左側選單找到 "Pages"
3. 在 "Source" 下選擇 "Deploy from a branch"
4. 選擇 "main" 分支和 "/ (root)" 資料夾
5. 點擊 "Save"
6. 等待幾分鐘，頁面會顯示您的網站網址

### 步驟 5：獲取網址
您的網址格式會是：
```
https://[您的GitHub用戶名].github.io/[repository名稱]/
```

例如：
```
https://johnsmith.github.io/investment-privacy/privacy-zh.html
https://johnsmith.github.io/investment-privacy/terms-zh.html
```

## 方法二：使用其他免費託管服務

### Netlify
1. 前往 https://www.netlify.com
2. 註冊帳號（可用 GitHub 登入）
3. 拖放您的 HTML 檔案到頁面
4. 獲得免費的 `.netlify.app` 網址

### Vercel
1. 前往 https://vercel.com
2. 註冊帳號（可用 GitHub 登入）
3. 連接您的 GitHub repository
4. 自動部署，獲得免費網址

## 常見問題

### Q: 我可以使用中文域名嗎？
A: 可以，但建議使用英文域名以確保兼容性。

### Q: 需要付費嗎？
A: GitHub Pages 完全免費，無需付費。

### Q: 可以自訂域名嗎？
A: 可以，GitHub Pages 支援自訂域名，需要購買域名並設定 DNS。

### Q: 如何更新內容？
A: 只需在 GitHub 上編輯檔案，更新會自動生效。

### Q: 需要 HTTPS 嗎？
A: GitHub Pages 自動提供 HTTPS，Apple 要求隱私政策必須使用 HTTPS。
