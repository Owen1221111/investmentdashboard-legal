# 投資儀表板 iOS App

## 專案簡介
這是基於現有Web版投資儀表板的iOS原生應用開發專案。

## 技術棧
- **平台**: iOS 15.0+
- **框架**: SwiftUI
- **數據存儲**: CloudKit + iCloud 同步
- **圖表**: Swift Charts
- **架構**: MVVM

## 專案結構
```
InvestmentDashboard-iOS/
├── README.md                 # 專案說明
├── Models/                   # 數據模型
├── Views/                    # SwiftUI 視圖
├── ViewModels/              # 視圖模型
├── Services/                # 服務層
├── Resources/               # 資源文件
└── Documentation/           # 相關文檔
```

## 功能特色
- 📊 多客戶投資組合管理
- 📱 6列響應式佈局 (iPad/iPhone 適配)
- 🔄 滑動卡片互動
- ☁️ iCloud 自動同步
- 📈 即時圖表更新
- 💾 離線數據緩存
