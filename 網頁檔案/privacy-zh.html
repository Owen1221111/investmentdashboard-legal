隱私權政策

生效日期：2025 年 1 月 1 日

最後更新日期：2025 年 12 月 2 日

簡介

感謝您使用 InvestmentDashboard（以下簡稱「本應用程式」或「我們的服務」）。我們非常重視您的隱私權。本隱私權政策說明我們如何收集、使用、儲存和保護您的個人資訊。

本應用程式由 Owen Liu（以下簡稱「我們」）開發和營運。使用本應用程式即表示您同意本隱私權政策的條款。

1. 資料收集

1.1 我們收集的資訊

本應用程式可能收集以下類型的資訊：

1.1.1 您主動提供的資訊

  • 客戶資料：客戶姓名、出生日期、聯絡資訊等
  • 資產資訊：投資組合、股票、債券、定存、保險保單等財務資料
  • 保單資訊：保險公司、保單號碼、保額、受益人等
  • 照片和文件：您上傳的保單照片、試算表文件等

1.1.2 自動收集的資訊

  • 裝置資訊：裝置型號、作業系統版本、唯一裝置識別碼
  • 使用資料：您如何使用本應用程式的資訊（功能使用頻率、錯誤記錄等）
  • 訂閱資訊：訂閱狀態、試用期開始/結束日期、付款記錄

1.2 我們不收集的資訊

  • 身份驗證資訊：我們不收集您的 Apple ID 或密碼
  • 支付資訊：所有支付處理由 Apple 處理，我們不儲存您的信用卡資訊
  • 位置資訊：我們不追蹤您的地理位置
  • 第三方帳號：我們不要求連結社交媒體或其他第三方帳號

2. 資料使用方式

2.1 主要用途

我們使用收集的資訊用於以下目的：

  • 提供服務：管理您的客戶和資產資料
  • 資料同步：透過 iCloud 在您的裝置間同步資料
  • 功能改進：分析使用情況以改善應用程式功能
  • 客戶支援：回應您的查詢和技術支援請求
  • 訂閱管理：管理您的訂閱狀態和試用期

2.2 資料處理

  • 所有資料處理均在您的裝置上或您的 iCloud 帳號中進行
  • 我們不會將您的資料傳輸到我們自己的伺服器
  • OCR 文字辨識功能使用 Apple 的 Vision 框架，在裝置上本地處理

3. 資料儲存和安全

3.1 儲存位置

  • 本地儲存：資料使用 Core Data 儲存在您的裝置上
  • 雲端同步：透過 Apple CloudKit 儲存在您的 iCloud 帳號中
  • 加密保護：所有資料均受 iOS 和 iCloud 的加密保護

3.2 安全措施

我們採取以下措施保護您的資料：

  • 使用 Apple 提供的安全框架（Core Data、CloudKit）
  • 資料傳輸使用 SSL/TLS 加密
  • 僅在您的 iCloud 帳號中儲存資料
  • 不將資料傳輸到第三方伺服器
  • 定期更新應用程式以修補安全漏洞

3.3 資料保留

  • 您的資料將保留在您的裝置和 iCloud 帳號中，直到您選擇刪除
  • 刪除應用程式後，本地資料將被移除
  • iCloud 資料的保留遵循 Apple 的 iCloud 政策
  • 您可以隨時在應用程式中刪除任何資料

4. 資料分享

4.1 我們不會分享您的個人資料

我們不會將您的個人資料出售、出租或分享給第三方，除非：

  • 法律要求：遵守法律義務、法院命令或政府要求
  • 您的同意：獲得您的明確同意
  • Apple 服務：使用 Apple 提供的服務（iCloud、App Store）

4.2 第三方服務

本應用程式使用以下服務：

Apple 服務

  • iCloud / CloudKit：用於跨裝置同步資料
  • StoreKit：用於處理訂閱付款
  • Vision Framework：用於 OCR 文字辨識（在裝置上本地處理）

這些服務受 Apple 隱私政策約束：https://www.apple.com/tw/privacy/

外部 API 服務

  • 匯率資料 API：用於取得即時匯率資訊
    • 用途：提供多幣別投資商品的匯率轉換功能
    • 資料傳輸：僅傳送貨幣代碼（如 USD、EUR、TWD 等），不包含任何個人資訊
    • 資料接收：僅接收公開的匯率數據
    • 頻率：僅在用戶主動請求時進行 API 呼叫
    • 隱私保護：API 請求不包含任何可識別用戶身份的資訊
    • 資料儲存：匯率資料僅暫存於裝置記憶體中，不會永久儲存
  • 股價資料 API：用於取得美股和台股即時股價
    • 用途：提供股票市值即時更新功能
    • 資料傳輸：僅傳送股票代碼（如 AAPL、2330.TW 等），不包含任何個人資訊
    • 資料接收：僅接收公開的股價數據
    • 頻率：僅在用戶主動點擊「更新股價」時進行 API 呼叫
    • 隱私保護：API 請求不包含任何可識別用戶身份的資訊

重要說明：

  • 所有 API 呼叫均為匿名請求，不會傳送任何個人資訊
  • API 服務提供商無法追蹤或識別個別用戶
  • 我們不會將您的投資組合資訊傳送給任何第三方
  • 匯率和股價資料為公開市場資訊，不涉及個人隱私

5. 您的權利

5.1 存取和控制您的資料

您擁有以下權利：

  • 存取權：隨時查看您儲存的所有資料
  • 修改權：編輯或更新您的資料
  • 刪除權：刪除任何或所有資料
  • 匯出權：匯出您的資料（未來版本將支援）
  • 訂閱管理：隨時取消訂閱

5.2 如何行使您的權利

  • 刪除資料：在應用程式中直接刪除個別項目
  • 管理 iCloud 資料：透過 iOS 設定 > [您的名稱] > iCloud 管理
  • 管理訂閱：iOS 設定 > [您的名稱] > 訂閱項目
  • 聯絡我們：如需協助，請透過下方聯絡資訊與我們聯繫

6. 兒童隱私

本應用程式不針對 13 歲以下兒童設計。我們不會故意收集 13 歲以下兒童的個人資訊。如果您發現我們收集了兒童的資訊，請立即聯絡我們，我們將盡快刪除該資訊。

7. 國際資料傳輸

由於我們使用 Apple iCloud 服務，您的資料可能儲存在 Apple 位於不同國家的資料中心。Apple 遵循國際資料保護標準，包括歐盟 GDPR 和其他適用法規。

8. Cookies 和追蹤技術

本應用程式不使用 Cookies、網頁追蹤器或類似技術。我們不追蹤您在其他網站或應用程式的活動。

9. 隱私政策變更

我們可能會不時更新本隱私權政策。重大變更時，我們將：

  • 在應用程式中通知您
  • 更新本文件頂部的「最後更新日期」
  • 在適當情況下，要求您重新同意

建議您定期查閱本政策以了解任何變更。繼續使用本應用程式即表示您接受更新後的政策。

10. GDPR 合規聲明（歐盟用戶）

如果您是歐盟居民，根據 GDPR，您享有以下額外權利：

  • 資料攜帶權：以結構化、常用格式接收您的資料
  • 限制處理權：在某些情況下限制我們處理您的資料
  • 反對權：反對處理您的資料
  • 撤回同意權：隨時撤回您的同意
  • 向監管機構投訴權：向資料保護監管機構提出投訴

11. 加州隱私權（CCPA）

如果您是加州居民，根據 CCPA，您享有以下權利：

  • 知情權：了解我們收集的個人資訊類別
  • 刪除權：要求刪除您的個人資訊
  • 選擇退出權：選擇不出售您的個人資訊（我們不出售個人資訊）
  • 不歧視權：行使隱私權時不受歧視

12. 聯絡資訊

如果您對本隱私權政策有任何疑問、疑慮或請求，請透過以下方式聯絡我們：

電子郵件：stockbankapp@gmail.com

回應時間：我們將在 7 個工作日內回應您的查詢
