## 📝 整理步驟

### 階段 1：備份
```bash
# 建立備份
mkdir _backup_$(date +%Y%m%d)
cp *.md _backup_$(date +%Y%m%d)/
```

### 階段 2：合併文件
1. 建立 `開發維護指南.md`（合併開發相關）
2. 建立 `功能使用手冊.md`（合併功能說明）
3. 建立 `問題排除手冊.md`（合併除錯文件）
4. 建立 `部署上架指南.md`（合併部署相關）

### 階段 3：整理法律文件
```bash
mkdir 法律文件
mv *Policy.md 法律文件/
mv *條款.md 法律文件/
```

### 階段 4：封存舊文件
```bash
mkdir _archive
mv PROJECT範本.md _archive/
mv NEW_PROJECT.md _archive/
mv superdesign-template讓ＣＬＵＡＤＥ參考/ _archive/
```

### 階段 5：刪除已合併的原始文件
**注意：** 確認合併無誤後再刪除！
